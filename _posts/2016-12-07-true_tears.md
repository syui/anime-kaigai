「私は涙を払ってくれたから」

それが最初に私を真の涙に導いたのです。 今私はこの同じラインを使って、この現実的なロマンスシリーズに皆さんを紹介したいと思います。 真の涙の遺産は、その時点で新しくリリースされた50のビジュアル小説のうち41点にランクされた非常に残念な視覚的小説の販売から始まりました。 ビジュアル小説の直後に、ゲームによく似たTrue Tearsのライブアクション映画が登場しました。 だから今、あなたはアニメが自然にビジュアル小説へのもう一つの適応であると考える必要がありますか？ 違う！

トゥルー・ティアーズのアニメ版は、視覚小説や映画（名前の横にある）とは何の関係もありません。 独自の独立したプロット、独自の独立した設定、独自の文字セットを持っています。 したがって、なぜTrue Tearsという名前を使用し、オリジナルのビジュアル小説の作成者にクレジットを与えるのかは、私の理解を超えて共通するものはほとんどありません。 しかし、私が理解しているのは、True Tearsは、LantisがKyoto AnimationからAir / Kanon / Clannadに類似のアニメタイプを制作しようとする試みだということです。 このレビューの目的は、上記のアニメとTrue Tearsを比較して批判することではありませんが、私は個人的にTrue Tearsがあればそのような比較にふさわしいと感じています。

多くの視聴者は、私たちが川を泣かせるような通常のタッチシーンなど、カノン（Kanon）やクラナド（Clannad）に似た真の涙の物語を多くの方法で見つけることができます。 しかし、物語の性質のために、半ダースのヒロインを扱う代わりに、我々は3つしかない。 したがって、キャラクターをよりよく発達させるために、より多くの時間を許す。 真の涙をハレームのアニメと呼ぶのは、すべての女性が私たちの主人公に"暴力を加える"ために誘惑されるかもしれませんが、私はこれを愛三角を開発するための不可欠な部分と見ています（そして、アニメには何もありません）。 アニメは通常のラブストーリーとはかなり異なるアプローチをとっており、言葉を慎重に使用することによって、この問題に本当のリアリズムがもたらされると言うことができます。 しかし、ある程度の深さを持つ他のほとんどの物語のように、真の涙には比喩的な比較が多く含まれていることに注意してください。 これらのメタファが何を意味するのかを理解することは難しくありませんが、一部の視聴者はそれらによって落胆する可能性があります。

すべての文字の関係が巧みに開発されています。 ほとんどのロマンスジャンルに共通する多層関係に加えて、若い世代のキャラクターと古い世代のキャラクターの相互作用、そしてそれらが互いにどのように影響するかもわかります。 主人公の両親は確かにこのジャンルのほとんどのアニメーションと比較してより中心的な役割を果たしました。 しかし、私が一番好きなのは、2人（または3人）の可能な恋人に直面したとき、若い男の子の心の完璧な描写です。 私は1つの理由で、瞬時に自分の考えを慎重に見た（私は2つのタイミングや何かではないが、彼の年齢の少年にとっては彼の行動は非常に論理的である）。 私は、私が彼の状況、特に衝撃的な出来事の後であれば、私はおそらく彼よりも混乱している（混乱していなくても）だろうと思う。

アニメーション賢明なのは確かにファンの期待に応えます。 これは特に、ショーの主要シーンに当てはまります。 アニメのタイトルから明らかなように、ここでは涙が真に異なって見えます。 真の涙の涙は非常に美しく、様々なキャラクターの頬を動かすと輝く塩辛い水が非常にリアルに見えます。 涙に加えて、キャラクターの動きの流動性、話すキャラクターの顔の詳細なシャドウ、3Dウォーキングシーンなどの特定の重要な側面を説明するために使用されるカメラの角度。 すべてが少しずつ追加され、それを知る前に、より低品質のアニメーションを見ることに苦労していることがわかります。 本当の涙と比べても優れたアニメーション品質のショーを見つけようとすると、まだ難しい時があります。

すばらしいアニメーション制作のように、True Tearsの音楽も敬遠されません。 音楽はBGMとOP / EDの両面で効果的に使用されていました。  BGMは、人々に涙目をかけるほどのシーンに触れるだけの気分を作り出します。  OPは、クラナドのファンがよく知っていなければならないeufoniusによるものです。 私は個人的には真の涙のOPがClannadよりはるかに優れていると感じています。  True Tearsのセイユは、ほとんどのアニメとは違って、私にはあまり関心をもたれなかったし、それを見ても何の役にも立たなかった。 メインキャストの中で最大の名前はおそらく奈良かおり（Nunnally in Code Geass、ユーレカはユーレカセブン、リネットはストライクウィッチーズ）。 このノートでは、私たち（アニメファン）に私のアニメのロマンスリストに永遠に残る素晴らしいロマンスシリーズをもたらしたLantisプロダクションクルーに本当に感謝したいと思います。

真の涙は、高校生の生活とストーリーの3人のヒロインとのインタラクションに焦点を当てた物語です。 ストーリーは、キャラクターの生活におけるロマンスとドラマに焦点を当てています。

私はストーリーラインが私が13エピソードのアニメのために見た最高のものだと思う。 そのような話にはちょうどいい長さだった。 アニメは多くのリアリズムを取り入れ、本当に愛の三角形のドラマを引き出します。 物語は、冒険の少ない道からの愛を見て、あなたの典型的なラブストーリーとは異なります。 すべてのメインガールズが主人公と恋に落ちるので、これはハーレムと呼ばれるかもしれませんが、より現実的で重大な角度から見ているので、通常のハーレムとは非常に異なります。 この番組では多くのドラマと意味を期待しています。 このショーは、「鶏が飛んでいる」という比喩的な使い方と奇妙でユニークな涙のテーマのために、このショーはちょっと謎めいています。 しかし、あなたが最初に理解していない場合でも、ひどいことはありません。

芸術は単に素晴らしいものでした。 そのような鮮やかさで背景や風景が描かれ、目には色が鮮やかで簡単でした。 メインキャラクターによるショーでの水彩画の使用は、ショーに非常に素晴らしいアニメーション感をも引き出します。 アニメで最高の芸術作品の一つ。

音はそうだった。 本当に目立ったのは声優だった。 特に4人の主人公。 彼らはキャラクターをうまく演奏し、アニメの感触に本当に貢献したと思う。 終わりに向かって雰囲気を作り出すのに役立つインサートが作られていて、私はOPの歌がこのアニメに非常に適していたと思う。 それは良いですが、最高のものではありません。

文字もとてもうまくいきました。 そのような短いアニメシリーズのための深さの良い契約をそれぞれ持っています。 特に、かなりの数の主人公がいたので、そのような短い範囲でこのような広範囲の文字をカバーするのはかなり良いです。 おそらくリードガールと男の2人の親友のようなサイドキャラクターにもう少し焦点を当てていたのはいいでしょうが、時間を考えればそれはまだOKです。 このキャラクターは、ロマンスアニメ、特にノエのあなたの通常のクリシェキャラクターではなく、このキャラクターを持つことで、ユニークで予測できないものになっていました。

私はそのロマンスとドラマのためにアニメを楽しんだ。 しかし、それはコメディ分野では欠けていた。 しかし、結果として、ドラマがかなりうまく引き出されたので大丈夫だと思います。 したがって、ストーリーは少し悲観的に見えるかもしれません。 スタートでもやや遅かったが、ストーリーが進むにつれてペースが上がるので、初めにこのアニメに困っているかもしれない人たちのために、辛抱強くやっている人たちのために。 しかし、私はキャラクターに共感することができ、時には非常に注目を集めるので、アニメは非常に意味があると思う。

全体的に、このアニメは見逃せません。 特にあなたがロマンスドラマのファンならば。 アニメは短い時間内にショーの多くの重要な要素を描写することができ、私はアニメがそれを見ているすべての人々に永続的な印象を残すと思う。 しかし、ストーリーやキャラクターに影響を与える過程で、忍耐力が必要な場合があります。 もっと明るい何かを探しているなら、おそらくあなたはこれをあなたが好きではないかもしれません。

真の涙は、（他のものの中でも）涙の珍しいテーマに基づいて、生涯、ロマンス、コメディ、ドラマのかなり典型的なスライスです。 それにもかかわらず、それは単なる別の「視覚的な小説」から「アニメの順応」です。 初めから、このタイプのアニメには、男性主人公（Shinichirou）がいて、彼とその周りとその物語のヒロイン（Noe、Hiromi＆amp; Aiko）が紹介されています。 しかし、物語が完全にどこにあるのか、それともどこに向かっているのかを、物事がある時点に達するまで伝えるのは本当に難しいです。 それはいくつかの主要な開発が起こったときですが、シリーズがその時点でほぼ半分になっていることは残念です。

ここでは、普通の高校少年少女のキャラクターの在庫があります。特に特別なものはありません。 しかし、それらのうちのいくつかは、彼らが目立つように独特の何かを持っています（Noe）。 しかし、キャラクターの関係は、単純な考え方の人以外はキャラクターを面白くするものです。 女の子たちは、彼らの本当の正直な気持ちが何であるかを完全に確信することができないので、ミックスに少しの混乱を加えます。

シリーズを見て、私は間違いなく、このアニメシリーズの最も優れた側面であると言うことができます。これは、すばらしい環境、よく描かれたキャラクター、 最も顕著な特徴は、柔らかい水の色を使用してペイントされたように見える背景であり、流体の前景アニメーションとシームレスによく混合されます。

音楽は様々なピアノとヴァイオリンの交響曲で構成されており、ショーのゆったりとした性質を活かしています。 それはまたいくつかのドラマのための気分を設定するには良い仕事をしますが、弱い対話はまったく役に立たない。

全体的な真の涙は、季節の修正を必要とするロマンスアニメ愛好家のために、素晴らしいロマンスアニメであることが判明しました。 ショーにはいくつかの驚くほど優れた側面があり、男性キャストのような大きな落とし穴を簡単に覆い隠すことができます。 初めのユーモアはこれを楽しいシリーズにするのに役立ちますが、いつものようにいくつかの大きなドラマを見ることを期待しています。 あなたがこのタイプのロマンスアニメに入っているなら、あなたがそれをチェックすることを確認してください。

真の涙はP.A. ソウルの独立系企業としてのデビューアニメ。 それは漫画、軽小説、視覚小説、何でも、それは元の生産だったに基づいていませんでした。 このため、スタジオは批評家のある割合で不注意であると考えられていましたが、これはシリーズの視聴者を増やすのに役立ちました。結局、大胆な動きでした。 ショーが進行し、人気が絶えず増加するにつれて、それはよく遊ばされた市場戦略になる。
それから、誰かが「もしそれが駄目なら元の制作に値するものは何だろう」と言うかもしれません。
興味深い点。 視聴者の期待に応えられないストーリーの作成にはメリットはありません。
幸運なことに、真の涙はその罠には入りません。 それは誰にとってもうまくいかないかもしれませんが、それは間違いなく精巧で慎重に行われた話です。
はい。 私は痛いところを突き刺したことをよく知っています。 以前の行が述べたものとは異なり、多くの人々は、その物語を、それが悪者であると主張するキャラクターの開発を責めにして、大きな失敗であると考えています。 非現実的なキャラクター、パターンストーリー、フラットパーソナリティーは、このアニメが受ける主要な批判の一部です。 それらは完全に間違っているわけではなく、もちろん私たち一人一人が自分の意見を持っていますが、これらの声明は小さな欠点に焦点を当てています。 意味：ストーリーラインの他の要素と、それらの「間違い」を説明して修正する小さな文字の対話によって、最終製品を見るときにすべての作品が一緒に視覚化され、これらの欠陥が見えなくなります。

真の涙をハレムと呼ぶのは誤解です。  Haremシリーズは、それに精通していない人にとっては、主な男性の鉛が女の子に囲まれているという話です。彼の状況を楽しんでいるので、主人公は彼の心を補うことができません。 純粋な心配のために。 真の涙の最初の2つのエピソードを見ている人は、このアニメがそれに関するものではないことに気付くでしょう。
ちょうどそのように、私は謙虚にMALのシリーズサマリーを考慮に入れないように頼んでいます。 明らかにそれを書いた人は、完全なシリーズを見ることはできませんでした（最初のエピソード全体を台無しにすること以外に）。

真の涙の強い属性はドラマであり、ロマンスはその重要なサイドキックである。 結局のところ、心温まる瞬間のない精神的な負荷シリーズは、見るのが面倒すぎるだけです。
ストーリーは、完全に理解するためにいくつかのポーズ・プレイ・アクションを必要とする隠喩やサブテキストによって強く表現された過去の状況をもとに、キャラクターの内的な葛藤を探っていきます。  「RaigomaruとJibetaの物語」は、生産者のポケットのエースを確信していた。
それの象徴性を強調するのはいいことです。 アニメは、両方のストーリー（実際のメインアニメーションと架空のライゴマルの両方）が独立して開発されるような構造であった。 しかし、それらは意味において合同であり、Raigomaru'sはキャラクターライフの出来事の結果によって絶えず影響を受ける。
ちょうどそのように、私はこのショーが完璧な人間的な方法で物語の出来事へのキャラクターの反応を表現する方法を感心しました。 ドラマシリーズは、感情的なパケットシーンが突然続いて、何も起こらないかのように日常的なイベントが穏やかに続き、口に酸っぱい味が残っているため、しばしば批判されることはよく知られています。  True Tearsはこのパターンから逃げ出します。RaigomaruのTaleが示すように、同じことが他のキャラクターにも適用されます。 彼女のバスケットボールの練習でのヒロミのパフォーマンス、ノエのベリーなど...

True Tearsのキャラクターを誤解するのは簡単です。 特に、誰かがそれを見たり、どこかでそれを読んだりするのを聞くと、 これらのレビューを読むことさえ、誰かがそれらを解釈する方法を変えるかもしれません。 うまくいけば、より良いものへ。
これらのキャラクターの性格は、私たちが慣れ親しんだものよりもずっと複雑で、単純に説明できないのは、実際に現実的なキャラクターを作ることに成功したからです。 しかし、このような進歩の課題は、すべてのキャラクターを同じレベルの複雑さに保ちすぎることです。そうでない場合、一部のキャラクターはフラットになり、他のキャラクターは濃くなります。 そして...これは起こったことです。 支持キャラクターは完全に無視されましたが、これはアニメの焦点のためにまだ受け入れられています。
3人の主人公を例にして、HiromiとShinichiroの背景ストーリーはうまく構成されており、その行動の理由を理解するには視聴者の側からより深いコミットメントが必要です。 一方、ノエのキャラクターは、自分の背景物語のほかに、他のものと比べて単純すぎて、さらなる開発をせずに、シリーズの終わりにプロデューサーの注目を集めました。 これは、偉大なドラマ/ロマンスが嫌悪感や心理的なものになるようになる（Huh、dejavúMakoto-san）人を楽しませたコントラストを作り出します。
それでも、全体的に、Hiromiのキャラクターで特別に行われた仕事は素晴らしかったです。 それは、私の個人的な意見では、キャラクターは10のうち9に値すると言いました。以前のように、最後の部分はこれらの間違いを隠しています。 彼らはそこにいることは知っていますが、ショーの楽しさには影響しません。

アニメーションとアートワークは本当に滑らかでした。 それはドラマなので、文字の表現が慎重にアニメ化され、それが起こったと予想されました。 また、雪片、葉、海水などの小さな細部の質について言及する価値はありますが、何よりも、このアニメは本当にその名前を表しました。 真の涙は私が今まで見た中で最高のTearsアニメーションの1つを持っていました。 恐ろしい「目の汗」アニメーションを持つシリーズを見つけることはまれではありません。 彼らがどのくらいよく描かれているかを見ることは驚きでした。
